# Energy-Accessibility-Analysis
The Global Energy Accessibility Analysis (1990–2025) project analyzes electricity access trends across multiple countries by comparing rural and urban populations. The aim is to identify growth patterns in electrification and highlight the urban–rural electricity access gap.

* Global Energy Accessibility Analysis (1990–2025) *

* Project Overview

This project analyzes global electricity accessibility trends by comparing rural and urban electricity access across multiple countries from 1990 to 2025. The objective is to identify growth patterns, highlight disparities between urban and rural electrification, and provide insights that support energy planning and sustainable development.
Interactive dashboards are created to visualize trends and enable data-driven decision-making for policymakers, researchers, and energy planners.

* Objectives

Analyze global electricity access trends from 1990–2025.
Compare rural and urban electricity accessibility across countries.
Identify growth patterns in electrification over time.
Highlight the urban–rural electricity access gap.
Provide insights through interactive data visualization dashboards.

* Dataset Information

The dataset contains global electricity access statistics with the following attributes:
Country – Name of the country
Year – Year of observation (1990–2025)
Rural Access (%) – Percentage of rural population with electricity access
Urban Access (%) – Percentage of urban population with electricity access
The dataset is sourced from publicly available international energy databases.

* Tools and Technologies

Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Creating calculated columns and measures
CSV Dataset – Data storage and preprocessing

* Data Processing Workflow

The system follows a structured data analysis workflow:

# Data Collection
Obtain electricity access datasets from global energy databases.

# Data Cleaning
Remove duplicate records
Handle missing values
Ensure valid data ranges

# Data Transformation
Convert data types
Standardize names
Prepare dataset for analysis

# Data Filtering
Filter years between 1990 and 2025
Validate electricity access percentages (0–100)
Merge two differnt tables using left join
Enable country-based filtering

# Data Analysis
Calculate key metrics including:
Rural growth
Urban growth
Urban–Rural electricity gap

# Data Visualization
Generate interactive dashboards including:
Trend analysis charts
Country comparison charts
Geographic maps
Key performance indicators

* Key Insights

The analysis highlights several important trends:
Electricity access has steadily increased globally over the last three decades.
Urban regions generally have higher electricity access compared to rural areas.
Several developing countries still experience significant rural electrification gaps.
Data visualization helps identify regions requiring targeted infrastructure development.
